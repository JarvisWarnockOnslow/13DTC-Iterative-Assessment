PROJECT TITLE: Movie Recommender
PURPOSE OF PROJECT: To recommend movie to the user based on other movies they like
VERSION or DATE: 4
HOW TO START THIS PROJECT: Start the main from the GUI class
AUTHORS: Jarvis Warnock
USER INSTRUCTIONS: Add movies to the program and then rate at least one. When
you press the recommend button the program will recommend movie to you that 
you have not rated that are in the same genre or have the same director 
as movies that you have rated above 6.

tl;dr:To get movies recommended make sure you have multiple movies that have not 
been rated, and that have the same genre or director as a movie that have a rating above 6
